# Spam-Email-Classification-using-Naive-Bayes
This project builds a machine learning model that classifies emails as Spam or Not Spam (Ham) using Natural Language Processing (NLP) and Naive Bayes algorithm. It demonstrates data preprocessing, feature extraction, model training, and evaluation.# Spam Email Classifier

This project classifies emails as Spam or Ham using Machine Learning and NLP.

## Features
- Text preprocessing
- Naive Bayes classification
- High accuracy for spam detection

## Tech Stack
- Python
- Scikit-learn
- Pandas

## How to Run
1. Clone the repository
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python spam_classifier.py
